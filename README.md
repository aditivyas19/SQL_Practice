# 💾 LeetCode SQL Solutions

Welcome to my **LeetCode SQL** repository!

This repository contains my solutions to SQL problems from LeetCode. I created it to strengthen my SQL fundamentals, improve query-writing skills, and prepare for technical interviews.

## 📚 Topics Covered

- Basic SELECT Statements
- Filtering Data
- Sorting Results
- Aggregate Functions
- GROUP BY & HAVING
- Joins
- Subqueries
- Common Table Expressions (CTEs)
- Window Functions
- CASE Statements
- Date Functions
- String Functions
- Advanced SQL Problems

## 📁 Repository Structure

Each SQL problem is stored in a separate `.sql` file.

Example:

```
LC_175_Combine_Two_Tables.sql
LC_176_Second_Highest_Salary.sql
LC_181_Employees_Earning_More_Than_Managers.sql
...
```

Each solution includes:
- Problem Number
- Problem Title
- A brief description of the approach
- SQL query
